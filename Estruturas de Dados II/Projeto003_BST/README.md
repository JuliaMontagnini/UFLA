# Projeto - Árvore Binária de Busca (BST)

Trabalho final desenvolvido para a disciplina de Estruturas de Dados II, com o objetivo de implementar uma Árvore Binária de Busca (BST) manualmente e compreender seus conceitos.

### Arquivos

**`No.py`**: Contém a classe responsável por representar os nós da árvore, armazenando o valor e as referências para os filhos esquerdo e direito.

**`ArvoreBinariaBusca.py`**: Contém a implementação da estrutura da BST e suas operações (inserção, busca, remoção, cálculo de altura, menor e maior valores, contagem de folhas e nós, e exibição de percursos).

**`App.py`**: Arquivo principal que executa os testes.

**`arquivo_de_saida.txt`**: Arquivo demonstrando os resultados obtidos na execução dos testes.

## Execução

- É preciso ter o Python 3 instalado (não são necessários pacotes externos).

No diretório contendo todos os arquivos, basta executar
   ```bash
   python3 App.py
